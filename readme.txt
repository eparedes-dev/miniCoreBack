Instrucciones parafuncionamiento de back:

1. Instalacion de Node.js en el equipo a ejecutar
2. Ejecutar en los comandos npm update
3. Ejecutar el comando npm install mysql2
4. Ejecutar el comando npm install express

Una vez realizado los pasos previos se debe configurar la conexion a la base de datos 

const connection = mysql.createConnection({
    host: 'Nombre del servidor',
    user: 'usuario que se usa', 
    password: 'en caso de tenerla colocarla',
    database: 'Nombre de la base de datos'
  });
